# Dependency Injection & Inversion of Control

## Dependency Injection
### 개념
- 의존 관계 주입 기능
- 객체를 직접 생성하는 것이 아니라 외부에서 생성한 후 주입

## Inversion of Control
### 개념
- 메소드나 객체의 호출작업을 개발자가 결정하는 것이 아니라, 외부에서 결정되는 것
- 기존
    1) 객체 생성
    2) 의존성 객체 생성
    3) 의존성 객체 메서드 호출
- IoC  
    1) 객체 생성
    2) 의존성 객체 주입
    3) 의존성 객체 메서드 호출
